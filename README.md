## 📋 Content Overview

Each assignment set contains multiple problems with varying difficulty levels:

- **A series**: Basic level problems (fundamental concepts)
- **B series**: Intermediate level problems (building on basics)
- **C series**: Advanced level problems (complex implementations)

## 🎯 Topics Covered

The assignments progress through core C programming concepts including:
- Basic input/output and variables
- Conditional statements (if-else)
- Loops (for, while, do-while)
- Arrays and strings
- Functions and recursion
- Pointers and memory management
- Structures and data structures
- File handling
- Algorithm implementations

## 💻 File Format

All files are standard C source code files (`.c` extension) that can be compiled and executed using any C compiler:

```bash
gcc filename.c -o output
./output
```

## 📚 Usage

Each file is a standalone program that can be compiled independently. Some files may have dependencies or build upon concepts from previous assignments.
